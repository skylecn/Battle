# Assets 资源目录说明

本文档描述 `Assets` 下各主要目录所存放的资源类型与用途。

---

## Scenes

**GameStart**：启动场景

---

## Scripts

**游戏与框架代码**：所有游戏功能。

- 代码结构说明见：**[Scripts/README.md](Scripts/README.md)**  

---

## Arts

**美术资源**：项目内使用的模型、材质与原型预制体。

| 子目录 | 内容说明 |
|--------|----------|
| **Materials** | 材质球 |
| **Models** | 模型 |
| **PrototypeMode** | 简模原型 |

---

## GameRes

**游戏业务资源**：运行时通过YooAsset加载的资源。

| 子目录 | 内容说明 |
|--------|----------|
| **Data** | 配置表 |
| **Prefabs** | 预制体 |
| **Scene** | 战斗场景：CityBattle。 |
| **UI** | FairyGUI资源。 |
